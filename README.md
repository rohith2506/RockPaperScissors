##### Description

Implementation of Rock Paper Scissors in Java


##### Setup

* Download the project into your custom location (USER_LOCATION)

* Compile
    1. cd $USER_LOCATION;
    2. mvn compile
    3. mvn package;

* Run
    1. java -cp target/rps-1.0-SNAPSHOT.jar com.strategygames.rockpaperscissors.Main
